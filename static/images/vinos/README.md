# Imágenes de vinos

Coloca aquí imágenes genéricas por tipo (JPG) para que el experto en vinos las muestre:

- **tinto.jpg** – vino tinto genérico  
- **blanco.jpg** – vino blanco genérico  
- **rosado.jpg** – vino rosado genérico  
- **espumoso.jpg** – vino espumoso genérico  
- **generico.jpg** – imagen por defecto cuando no hay coincidencia por tipo  

Puedes usar fotos libres de Unsplash o Pexels.  
Opcionalmente, añade `{vino_key}.jpg` (ej. `marques_de_riscal.jpg`) para vinos concretos.

Endpoint de stock

GET /api/bodega/stock

Devuelve la lista de vinos con cantidad actual. Formato JSON apto para consumo externo.

Cabecera obligatoria:

  • X-API-Token: <tu_token>

Ejemplo de solicitud (cURL):

curl -X GET "https://tu-dominio.com/api/bodega/stock" \
  -H "X-API-Token: TU_TOKEN_AQUI"

Ejemplo de respuesta:

{
  "stock": [
    {
      "id": "uuid-botella",
      "vino_nombre": "Marqués de Riscal Reserva",
      "cantidad": 3,
      "anada": 2020,
      "ubicacion": "Estantería 2",
      "tipo": "tinto"
    }
  ],
  "total_entradas": 1
}

Webhook (opcional)

Si configuras una URL de webhook, al añadir, modificar o eliminar botellas en Mi Bodega enviaremos un POST a esa URL con el cuerpo:

{
  "event": "bodega.updated",
  "stock": [ ... ]
}

El array stock tiene el mismo formato que la respuesta de GET /api/bodega/stock.

Ejemplo para registrar una venta:

curl -X POST "https://tu-dominio.com/api/adaptador/venta" \
  -H "Content-Type: application/json" \
  -H "X-API-Token: TU_TOKEN_AQUI" \
  -d "{\"vino_nombre\":\"Marqués de Riscal Reserva\",\"cantidad\":1}"

Guía para Desarrolladores

Cuando activas el modo firmado, VINO envía la cabecera X-Vino-Signature para verificar autenticidad. Firma: hmac_sha256(key=webhook_secret, msg=timestamp + "." + payload).

Usa el cuerpo JSON exacto recibido, sin modificar espacios ni reserializar el payload antes de calcular la firma.

Node.js

const crypto = require('crypto');

function verifyVinoSignature(secret, timestamp, rawBody, signatureHeader) {
  const signedPayload = `${timestamp}.${rawBody}`;
  const expected = 'sha256=' + crypto
    .createHmac('sha256', secret)
    .update(signedPayload, 'utf8')
    .digest('hex');
  const expectedBuffer = Buffer.from(expected, 'utf8');
  const receivedBuffer = Buffer.from(signatureHeader || '', 'utf8');
  if (expectedBuffer.length !== receivedBuffer.length) return false;
  return crypto.timingSafeEqual(expectedBuffer, receivedBuffer);
}

Python

import hmac, hashlib

def verify_vino_signature(secret, timestamp, raw_body, signature_header):
    signed_payload = f"{timestamp}.{raw_body}"
    expected = "sha256=" + hmac.new(
        secret.encode(), signed_payload.encode(), hashlib.sha256
    ).hexdigest()
    return hmac.compare_digest(expected, signature_header or "")
